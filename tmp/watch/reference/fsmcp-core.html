FsMcp.Core Namespace

Type/Module Description

Content (Module)

Convenience constructors for Content.

Content (Type)

Content carried by tool results, resource reads, and prompt messages.

McpError

Errors that can occur in the FsMcp toolkit.

McpMessage

A message in an MCP conversation.

McpRole

The role of a participant in an MCP conversation.

PromptArgument

An argument definition for a prompt.

PromptDefinition

A prompt definition with a name, description, argument list, and handler.

ResourceContents

Resource contents — either text with a MIME type or raw binary data.

ResourceDefinition

A resource definition with a URI, name, description, MIME type, and handler.

ResultExtensions

Helper to unwrap Result or throw with a descriptive message. Use instead of `Result.defaultWith failwith` which doesn't compile with ValidationError.

Serialization

JSON serialization support for FsMcp domain types. Provides custom converters matching the MCP wire format.

ToolDefinition

A tool definition with a name, description, optional input schema, and handler.

Types

Empty module to ensure this file compiles as a valid F# source.

Validation

Validated identifier types with smart constructors. Each type is a single-case DU that guarantees validity at construction time.

ValidationError (Module)

ValidationError (Type)

Validation errors for domain type construction.

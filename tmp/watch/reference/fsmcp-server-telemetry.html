Telemetry Module

Namespace: FsMcp.Server
Assembly: FsMcp.Server.dll

Built-in telemetry middleware using System.Diagnostics.Activity. Compatible with OpenTelemetry, Application Insights, and any ActivityListener.

Types

Type Description

MetricsCollector

Metrics collector that tracks request counts and durations. Keeps only the last 1000 durations per method to prevent memory leaks.

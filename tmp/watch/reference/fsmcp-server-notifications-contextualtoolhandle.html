ContextualToolHandle Type

Namespace: FsMcp.Server
Assembly: FsMcp.Server.dll
Parent Module: Notifications
Base Type: obj

Per-tool handler storage. Each ContextualTool.define call returns a ToolDefinition whose Handler closes over its own context-aware handler, avoiding a process-global registry that causes test isolation issues. A contextual tool definition that captures its context-aware handler in a closure (no global registry — test-safe).
